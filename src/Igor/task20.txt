Переменная x объявлена только в цикле for, поэтому код не скомпилируется (java выдаёт ошибку).
Чтобы код скомпилировался, изначально надо объявить переменную x.
void bar() {
        int x;
        int[] nums = {1, 2, 3};
        for (x = 0; x < nums.length; x++) {
            x += nums[x];
        }
        System.out.println(x);
    }
Ответ 6.
